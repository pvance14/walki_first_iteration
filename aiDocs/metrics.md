# MVP Metrics Targets

Updated: February 24, 2026

- Quiz completion rate: `Quiz Complete` / quiz starts, target `>70%`
- Demo engagement: `Demo Engagement` event with `durationBucket`, target average `>3 minutes`
- Motivation actions: `Get Motivation Click`, target `>3 clicks per session`
- Waitlist conversion: `Waitlist Signup` / quiz completers, target `>20%`
- Social sharing: `Share` events / quiz completers, target `>5%`

Notes:
- Analytics events must not include PII (no email addresses or free-text content).
- Waitlist email addresses are collected only through waitlist submission endpoint/local storage fallback.
