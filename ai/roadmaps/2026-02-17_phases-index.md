## Navigation Guide

### How to Use These Docs
1. **Before starting a phase:** Read the plan doc for detailed implementation guidance
2. **During the phase:** Use the roadmap doc to track progress (check off tasks)
3. **After completing:** Move both docs to `ai/roadmaps/complete/` and update `ai/changelog.md`

### Document Naming Convention
- Plans: `YYYY-MM-DD_phase-N-name-plan.md`
- Roadmaps: `YYYY-MM-DD_phase-N-name-roadmap.md`

### Status Values
- Not Started
- In Progress
- Blocked (with reason)
- Completed
